Для работы майл маркетинга:
 Проверить файл .env на правильно указанные данные почтового сервера, логин пароль, порт, хост
 Проверить правильно ли указана база данных. (ну такой себе пунктик, пусть лучше ему быть чем не быть здесь)
 Что бы велись логи, в папке config должен быть файл logining.php, в массиве channels должен быть такой кусок кода:
        'logInfo' => [
            'driver' => 'single',
            'path' => storage_path('logs/logInfo.log'),
            'level' => 'debug',
        ],
 вставишь, кастомные логи должны заработать.
